Case Study:

Depot was a simple 2 person eCommerce Project built with the use of our basic knowledge of PHP. It is a website wherein users are able to login and "purchase" an item based from the game Team Fortress 2. The items were stored inside a database using MySQL. The major hurdle with this project is that it was developed during the start of the pandemic, as almost all of us in class including the professor were adjusting to the new normal. Depot is therefore one the turning points in my academics in which I learned how to handle projects and the team through pure online interaction.
